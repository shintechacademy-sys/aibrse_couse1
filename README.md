# Python Course - Programming Fundamentals to Mini Project

## 📌 Giới thiệu khóa học

Đây là khóa học Python dành cho người mới bắt đầu, đi từ **cơ bản đến xây dựng pipeline mini project**.  
Khóa học tập trung vào tư duy lập trình, thực hành nhiều bài tập (homework), và phát triển khả năng tự giải quyết vấn đề.

---

## 🎯 Mục tiêu khóa học

Sau khóa học, học viên có thể:

- Hiểu và sử dụng Python cơ bản thành thạo
- Viết chương trình với cấu trúc rõ ràng
- Sử dụng hàm để chia nhỏ bài toán (decomposition)
- Làm việc với các cấu trúc dữ liệu phổ biến
- Hiểu lập trình hướng đối tượng (OOP)
- Có tư duy debug và xử lý lỗi
- Biết xây dựng pipeline xử lý dữ liệu đơn giản
- Làm việc với API cơ bản và I/O schema
- Hoàn thành mini project end-to-end

---

## 🧠 Nội dung khóa học

### 1. Python & Coding Basics

- Biến, kiểu dữ liệu
- Câu lệnh điều kiện (`if/elif/else`)
- Vòng lặp (`for`, `while`)
- Toán tử và biểu thức
- Input / Output cơ bản

### 2. Hàm & Decomposition

- Định nghĩa hàm (`def`)
- Tham số và giá trị trả về
- Tư duy chia nhỏ bài toán
- Scope (phạm vi biến)

### 3. Data Structures

- List, Tuple, Set, Dictionary
- Cách lựa chọn cấu trúc dữ liệu phù hợp
- Xử lý dữ liệu cơ bản

### 4. Class & OOP

- Class và Object
- Constructor (`__init__`)
- Method và thuộc tính
- Encapsulation, Inheritance cơ bản

### 5. Debug Mindset

- Đọc lỗi (error reading)
- Debug từng bước
- Tư duy kiểm tra logic

### 6. Python Pipeline

- Xử lý dữ liệu theo từng bước
- Thiết kế luồng xử lý (input → process → output)

### 7. API Basics & I/O Schema

- Khái niệm API
- Request / Response cơ bản
- JSON format
- Input/Output schema rõ ràng

### 8. Mini Project

- Xây dựng pipeline hoàn chỉnh
- Kết hợp toàn bộ kiến thức đã học
- Thiết kế chương trình có cấu trúc

### 9. Exam #1 + Review

- Ôn tập toàn bộ kiến thức
- Làm bài kiểm tra tổng hợp
- Review và cải thiện code
